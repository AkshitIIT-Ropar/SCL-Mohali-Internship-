# SCL-Mohali-Internship-
Study of Indigenous Flight Controller using SHAKTI Yamuna RISC-V32 SoC
 
 Conducted a technical study of the SHAKTI Yamuna RISC-V32 SoC for UAV flight controller applications, focusing on processor architecture, SDK, and peripheral interfaces.
Evaluated the software stack required for flight controllers, including RTOS, hardware abstraction layers (HAL), and communication protocols such as SPI, I²C, UART, and PWM.
Investigated the roadmap and technical challenges involved in porting an STM32-based flight controller software ecosystem to an indigenous RISC-V platform.
